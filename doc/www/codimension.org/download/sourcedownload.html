Download Codimension Source Code

The Codimension source code is split between a few GitHub repositories:

  • brief python parser: extracts what is inside a python file - classes, functions, global variables etc. The parser is written mostly in C.
  • control flow parser: extracts the content of a python file in terms of fragments which are used to generate a fowchart-like diagram. The parser is written mostly in C++.
  • Codimension IDE: that's the GUI which is written in Python.

brief python parser

  • GitHub repository
  • GitHub releases area where the source archives could be downloaded
  • Cloning the brief python parser repository: 
    $ cd
$ git clone https://github.com/SergeySatskiy/cdm-pythonparser.git
$ cd cdm-pythonparser

control flow parser

  • GitHub repository
  • GitHub releases area where the source archives could be downloaded
  • Cloning the flow parser repository: 
    $ cd
$ git clone https://github.com/SergeySatskiy/cdm-flowparser.git
$ cd cdm-flowparser

Codimension IDE

  • GitHub repository
  • GitHub releases area where the source archives could be downloaded
  • Cloning the flow parser repository: 
    $ cd
$ git clone https://github.com/SergeySatskiy/codimension.git
$ cd codimension
