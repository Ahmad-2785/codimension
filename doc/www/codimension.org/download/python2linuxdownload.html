Python 2 Linux Download and Installation (obsolete)

Note: Python 2 versions are not supported anymore.

The latest Codimension IDE version for Python 2 is 3.0.2.

The installation uses Linux packaging system.

Codimension packages are built for a few platforms (64 bit versions only). Note, that the packages were tested on Ubuntu only, the other platforms received a sanity check only.

  • Centos
  • Fedora
  • Debian
  • Ubuntu

On all the platforms you need to install at least three packages:

  • brief python parser: binary python 2 extension module
  • control flow parser: binary python 2 extension module
  • Codimension IDE: python 2 package

Please see details below for each the supported platform.

Centos Download and Installation

Download the latest packages:

  • brief python parser cdm-pythonparser-2.0.1-1.el7.centos.x86_64.rpm
  • control flow parser cdm-flowparser-1.0.1-1.el7.centos.x86_64.rpm
  • Codimension IDE codimension-3.0.2-2.noarch.rpm

Run in console:

$ sudo yum install <downloaded packages>

Run Codimension:

$ codimension

Fedora Download and Installation

Download the latest packages:

  • brief python parser cdm-pythonparser-2.0.1-1.fc23.x86_64.rpm
  • control flow parser cdm-flowparser-1.0.1-1.fc23.x86_64.rpm
  • Codimension IDE codimension-3.0.2-2.noarch.rpm

Run in console:

$ sudo dnf install <downloaded packages>

Run Codimension:

$ codimension

Debian Download and Installation

Download the latest packages:

  • brief python parser cdm-pythonparser_2.0.1-Debian.jessie_amd64.deb
  • control flow parser cdm-flowparser_1.0.1-Debian.jessie_amd64.deb
  • Codimension IDE codimension_3.0.2-Debian.jessie_all.deb
